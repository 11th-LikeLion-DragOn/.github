## 🐉 프로젝트 소개 
> 새해맞이 목표를 세우는 당신을 위한 챌린지 서비스 <b>drag-on </b> <br />
> 유연한 목표 달성 방식과 친구 간 달성률 공유를 통해 지치지 않는 챌린지 성공을 경험해요🔥
